# Tennis

## 🎾 개요

- 테니스 모임 서비스의 서버를 설계하고 구축합니다.
- 유지보수성을 고려하여 객체지향설계에 따라 개발하고, 테스트코드를 작성합니다.
- 다음 기능들을 구현합니다.
    - 회원 가입 / 로그인 / 로그아웃
    - 도메인 데이터 CRUD
    - 페이징 및 정렬
    - 예외 처리
    - 테스트
